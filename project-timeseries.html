Time Series Project

MediaTek Stock Time Series Analysis

A final report focused on forecasting and model diagnostics using MediaTek stock data and a workflow built around ARIMA-style time series analysis.

Overview

This project analyzes daily stock price data for MediaTek (2454) with the goal of understanding stationarity, selecting a suitable model, and evaluating forecast performance on a held-out period.

Methods

  • Log return transformation
  • ADF unit root testing
  • ARIMA model selection and comparison
  • Residual diagnostics and adequacy checks
  • Out-of-sample forecast evaluation

Results

The report demonstrates a structured modeling pipeline: transformation to improve stationarity, formal testing, candidate model comparison, residual assessment, and forecast evaluation using a final testing window.

Project Summary

  • Topic: Financial time series
  • Focus: Forecasting and diagnostics
  • Tools: R, statistical modeling
