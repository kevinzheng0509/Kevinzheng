Regression Project

ACT and GPA Prediction

A regression-focused coursework project centered on model interpretation, interval estimation, and prediction in an educational dataset.

Overview

This project studies the association between ACT scores and GPA through simple linear regression, emphasizing interpretation rather than only numerical output.

Methods

  • Simple linear regression fitting
  • Confidence intervals for mean response
  • Prediction intervals for individual response
  • Hypothesis testing for regression coefficients
  • Model interpretation in applied context

Why It Matters

This project reflects foundational training in regression analysis and shows the ability to connect statistical output with substantive explanation, a skill that is central to future research work.
